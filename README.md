## About This Project

This repository contains the source code for my **personal portfolio website**, a minimalist and responsive single-page application built entirely with vanilla technologies — no frameworks, no build tools, no dependencies.

The site presents my professional background, technical expertise, and project portfolio in a clean, recruiter-friendly format with full bilingual support (Portuguese/English).

**[View Live Site →](https://reiquileut.github.io/portfolio/)**

## Site Features

- **Bilingual (PT/EN)** — One-click language toggle with persistent preference via `localStorage`
- **Fully Responsive** — Adapts from desktop to mobile using CSS Grid and Flexbox
- **Zero Dependencies** — Pure HTML5, CSS3, and vanilla JavaScript
- **Modern Typography** — [Source Serif 4](https://fonts.google.com/specimen/Source+Serif+4) + [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts
- **Automated Deployment** — GitHub Pages via GitHub Actions on every push to `main`

## Running Locally

No build step required:

```bash
# Open directly in your browser
start index.html            # Windows
open index.html             # macOS

# Or use any static server
python -m http.server 8000
npx serve .
```
